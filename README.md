# 🖥️ Yaya Frontend — Sistema de Caja (POS)

Interfaz moderna, rápida y optimizada para el uso diario en un maxikiosco.  
Diseñada para trabajar con teclado, impresora térmica y flujos reales de venta.

---

## 🚀 Tecnologías principales
- Next.js 14 (App Router)
- React + Hooks
- CSS Modules
- Fetch API
- Diseño dark mode

---

## 💸 Funcionalidades POS
### ✔ Búsqueda rápida de productos
- Por nombre
- Por código
- Navegable con teclado

### ✔ Carrito de venta
- Incrementar / decrementar cantidades
- Eliminar productos
- Cálculo automático del total

### ✔ Medios de pago
Popup dinámico según el método:
- Efectivo (monto recibido + vuelto)
- Débito (autorización + operación)
- Crédito (cuotas + interés)
- Transferencia
- Mercado Pago
- Fiado (selección de cliente)

### ✔ Ticket térmico
- Formato clásico 58/80mm
- Texto plano
- Impresión automática
- Vista previa integrada

### ✔ Navegación con teclado
- ENTER para confirmar
- ESC para cerrar popups
- Autofocus inteligente
